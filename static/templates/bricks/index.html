Ever since I was a kid, I loved playing with Legos. I liked to build everything from baseball parks to starships. A few months ago, I decided to take my passion for these little plastic toys to the next level and use them to help sharpen my programming skills. Lo and behold, this is the result! I'm currently entering all of my Lego bricks into a Django database (which is connected to PostgreSQL), and the charts below showcase my collection by brick type and color. It's still a work-in-progress as there's many bricks that still need to be added. (Also, I'm still working on the front-end of this page.) But updates should be made regularly and each chart will update whenever a new brick has been added. If you have any questions or comments, please email me at hello@soupp.tv. Thanks so much for visiting. I hope you enjoy Project Lego!
